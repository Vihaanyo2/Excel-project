superstore Excel Dashboard Project
Project Overview
This project analyzes Superstore sales data using Excel dashboards and data analysis techniques to identify sales trends, profit performance, and customer behavior.

---

## Business Requirement
The objective of this project was to:
- Analyze regional sales performance
- Identify high-profit and low-profit categories
- Track monthly sales trends
- Improve decision-making using dashboard insights

---

## Tools & Technologies Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Conditional Formatting
- Slicers
- Formulas (SUMIFS, VLOOKUP, IF, COUNTIF)

---

## Dataset Information
Dataset contains:
- Orders
- Sales
- Profit
- Category
- Region
- Customer Segment

---

## Dashboard Features
- Sales KPI Cards
- Region-wise Sales Analysis
- Category Profit Analysis
- Monthly Trend Analysis
- Interactive Filters using Slicers

---

## Challenges Faced
- Cleaning missing and duplicate data
- Handling incorrect date formats
- Creating dynamic dashboards
- Managing large datasets efficiently

---

## Key Insights
- Technology category generated highest profit
- West region had maximum sales
- Some categories had high sales but low profit
- Consumer segment contributed most revenue

---

## Recommendations
- Focus marketing on profitable categories
- Improve low-performing regions
- Reduce discount dependency
- Increase inventory for top-selling products
